🚀 Getting Started
For C++ Projects (Banking System / Operation In Array)

Clone the repository:

bash   git clone https://github.com/ABINASH123000/High-School-Project.git

Navigate to the desired project folder:

bash   cd "Banking System"
   # or
   cd "Operation In Array"

Compile the C++ file:

bash   g++ filename.cpp -o output

Run the program:

bash   ./output
For Nepal Tourism Website

Navigate to the website folder:

bash   cd "Nepal Toursim Website"

Open index.html in any web browser.
